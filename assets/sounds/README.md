# Sound Files

## Alarm Sound Setup

To add sound to the loud alarm feature:

1. **Download an alarm sound file** (MP3 format recommended)
2. **Rename it to `alarm.mp3`**
3. **Place it in this directory** (`assets/sounds/alarm.mp3`)

### Recommended Alarm Sounds:
- Emergency siren sound
- High-pitched alarm tone
- Police/ambulance siren
- Fire alarm sound

### File Requirements:
- **Format**: MP3
- **Duration**: 10-30 seconds (will loop)
- **Size**: Under 1MB for fast loading
- **Quality**: Clear and loud

### Alternative Setup:
If you don't have an alarm sound file, the app will show an error message but continue to work. You can add the sound file later.

### Testing:
After adding the alarm.mp3 file, restart the app and test the "Loud Alarm" button in the Toolkit screen.
